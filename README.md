# 🏃 Word Collector — English Vocabulary Game

An interactive browser-based game built for school students to learn English grammar in a fun, engaging way — inspired by Subway Surfer-style gameplay.

---

## 🎮 How to Play

- A boy runs on screen — word tiles scroll from right to left
- **Tap correct words** → earn **+5 points**
- **Tap wrong words** → lose **-1 point**
- Reach the **target score** before time runs out to unlock the next level
- Progress through **3 levels** in each section

---

## 📚 Sections Covered

| Section | Topics |
|---|---|
| 📦 Nouns | Common, Proper, Abstract |
| ⚡ Verbs | Action, Past Tense, Linking |
| 🎨 Adjectives | Size, Emotions, Comparatives |
| 🚀 Adverbs | Manner, Time, Place |
| 🔗 Prepositions | Place, Time, Direction |
| 🔀 Conjunctions | Coordinating, Subordinating, Mixed |

---

## 🏆 Scoring System

| Action | Points |
|---|---|
| Correct word collected | +5 |
| Wrong word tapped | -1 |
| Level 1 target | 20 pts |
| Level 2 target | 30 pts |
| Level 3 target | 40 pts |

---

## ⚙️ Features

- 6 grammar sections × 3 levels = **18 levels** total
- Time limit per level with countdown warning
- Score progress bar
- Animated running character
- Word tiles on 3 staggered lanes — no overlap or clutter
- Instant feedback on every tap
- Fully browser-based — no installation needed

---

## 🛠️ Built With

- HTML5
- CSS3 (animations, responsive layout)
- Vanilla JavaScript (requestAnimationFrame for smooth gameplay)
